## 📌 1. Descripción de la aplicación

**Generador APA 7 PRO** es un programa para gestionar y exportar referencias bibliográficas en formato **APA 7** de forma rápida y sencilla.

Permite:
- Cargar referencias desde archivos bibliográficos
- Visualizarlas en una lista
- Eliminar referencias seleccionadas
- Exportar un documento Word listo para usar

---

## 🖥️ 2. Interfaz de usuario

La aplicación cuenta con un diseño simple y funcional.

### 🔝 2.1 Barra superior
- **Nombre de la aplicación:** “Generador APA 7”
- **Botón “Acerca de”**: muestra información del desarrollador y contacto

---

### 🧩 2.2 Panel principal

#### 🔘 Botones principales

1. **Cargar RIS/NBIB**
   - Permite abrir archivos `.ris` o `.nbib`
   - Importa automáticamente las referencias

2. **Eliminar referencias**
   - Borra las referencias seleccionadas de la lista

3. **Exportar Word**
   - Genera un archivo:
     ```
     bibliografia_APA7.docx
     ```
   - Incluye todas las referencias en formato APA 7
   - Añade hipervínculos de DOI si existen

## ⚙️ 3. Cómo usar la aplicación

### 📥 3.1 Cargar referencias
1. Haz clic en **Cargar RIS/NBIB**
2. Selecciona uno o varios archivos `.ris` o `.nbib`
3. Las referencias se añaden automáticamente a la lista

📌 Nota: Las referencias se convierten automáticamente a formato **APA 7**

---

### 👀 3.2 Revisar referencias
- Las referencias aparecen en la lista central
- Puedes desplazarte y comprobar títulos y años

---

### ❌ 3.3 Eliminar referencias
1. Selecciona una o varias referencias (Ctrl para múltiples)
2. Pulsa **Eliminar referencias**
3. Confirma la acción

✔ Útil para limpiar la lista antes de exportar

---

### 📤 3.4 Exportar a Word
1. Haz clic en **Exportar Word**
2. Se genera el archivo:
3. Incluye por cada referencia:
- Autor(es)
- Año de publicación
- Título del artículo
- Revista (en cursiva si aplica)
- DOI con hipervínculo (si existe)
